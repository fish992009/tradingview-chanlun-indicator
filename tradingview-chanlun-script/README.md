# 缠论TradingView买卖点指标

基于缠论核心概念开发的TradingView Pine Script指标，用于识别分型、笔、线段和买卖点。

## 功能特点

- 自动识别顶分型和底分型
- 绘制笔和线段
- 识别价格中枢
- 标记各类买卖点
- 支持警报功能

## 使用方法

1. 在TradingView中打开Pine编辑器
2. 复制粘贴代码
3. 添加到图表并调整参数
4. 结合其他指标验证信号

## 参数说明

- `观察周期`: 分析的历史数据长度
- `最小笔长度`: 构成笔所需的最小K线数量
- `显示控制`: 可选择显示/隐藏各元素
